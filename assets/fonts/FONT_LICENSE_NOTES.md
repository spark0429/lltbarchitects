# Font Licensing Notes

## Pretendard

This website uses Pretendard for all typography.

Pretendard is loaded in index.html from the official jsDelivr distribution:

https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/static/pretendard.min.css

The Pretendard project is distributed under the SIL Open Font License. This license allows use, modification, embedding, and redistribution, including commercial website use, as long as the font is not sold by itself.

## Removed Font Dependencies

Futura PT, KoPub Dotum, and other previous fallback choices are no longer referenced by the project CSS. If a different commercial font is added later, confirm and keep a valid webfont license before publishing.
